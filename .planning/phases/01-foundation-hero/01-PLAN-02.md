# Wave 2 — Hero Section + Framer Motion Animation

**Goal:** Extract the hero markup into a proper `Hero.tsx` section component, introduce the `FadeIn.tsx` reusable scroll-animation wrapper, implement the Framer Motion stagger entrance animation for the five hero elements, and seed typed content in `lib/data/hero.ts`. Update `app/page.tsx` to use the new Hero component.

**Prerequisite:** Wave 1 complete — `next dev` runs clean, dark background verified, fonts self-hosted, static hero shell visible.

**Requirements covered:** HERO-01, HERO-02

    Element 2 — Hero name (delay 100):
    `<motion.h1 {...anim(100)} className="font-bold text-[#f5f5f5] mb-2" style={{ fontSize: 'clamp(48px, 6vw + 8px, 72px)', lineHeight: 1.1, letterSpacing: '-0.02em' }}>{heroData.name}</motion.h1>`

    Element 3 — Role descriptor (delay 200):
    `<motion.p {...anim(200)} className="text-lg font-light text-[#a3a3a3] tracking-wide mb-4">Freelance Software Engineer</motion.p>`
