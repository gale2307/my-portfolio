## Logo approach

Use a **text-based fallback approach** since real client logos may not be available:

- `lib/data/clients.ts` holds client names and a `logo?` optional path
- If no logo path, render a styled text block as a placeholder (company name in Geist Mono, muted, bordered box)
- If a logo path exists (e.g. `public/clients/acme.svg`), render an `<img>` with `filter: grayscale(1) opacity(0.6)`
- This way the section looks intentional with or without real logos
